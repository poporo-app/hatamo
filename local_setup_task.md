## 依頼内容

ローカル環境構築

1. docker-composeのセットアップ
- バックエンド: go *フレームワークを使用して下さい(何を使うかはお任せします)
- フロントエンド: Next.js15(SSR)
- データベース: MySql(latest)
- キャッシュ: Redis

2. フロントエンドのセットアップ
- tailwindCssを入れる
- Zustandを入れる
- jestを入れる
- StoryBookを入れる

3. バックエンドのセットアップ
- 簡単なHelloWorldを返すだけのRestApiを実装

4. バックエンドとフロントエンドはそれぞれディレクトリを切ってファイルを配置する
バックエンド: backend
フロントエンド: frontend
