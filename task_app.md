## 依頼内容

0. ドメイン分割は変更なし
1. 事業者側に作成した事業者登録ページへの導線をヘッダーに置いて下さい。
2. 追加した機能にテストコード実装(必要なら)
3. 動作確認（エラーが出ない事を確認する）
4. コミット&push
5. プルリク作成

## デザイン

シックで大人っぽいデザイン

## 仕様
異次元コミュニティ「HATAMOマッチングサービス」システムの 画面遷移図（簡易版テキスト）
1. ユーザー（利用者）視点 : localhost:3001
ログイン・新規登録ページ
トップページ → 事業者一覧 / 検索
事業者詳細ページ → サービス内容 / 料金 / 申込ボタン
申込フォーム → 利用者情報入力 / 同意チェック
決済画面（Stripe） → 決済完了
完了画面 → 申込完了メッセージ
2. 事業者視点 localhost:3002
ログイン・新規登録ページ 
事業情報入力 / 必要書類アップロード
審査待ち画面 → 「運営審査中」ステータス
審査通過メール通知
案件登録(審査通過後)
決済画面（Stripe/アルファノート） → 月額料金支払い
管理画面トップ
　・案件一覧（利用者からの申込状況）
　・メッセージ通知（メール/LINE連動）
　・売上明細確認
3. 運営（管理者）視点 lcalhost:3003
管理者ログイン画面
スポンサー審査画面 → 承認 / 否認 / 再申請依頼
掲載管理画面 → 事業者リスト / 公開・非公開設定
申込管理画面 → 利用者申込一覧 / ステータス確認
精算画面 → 手数料控除後の振込額自動計算 / 明細PDF生成
:南京錠: 安全・法務導線
各フォームに reCAPTCHA
各購入・申込画面に キャンセル・返金規約同意チェック
フッターに 利用規約 / 特商法表示 / プライバシーポリシー
