## 依頼内容

1. ユーザー側で実装した新規登録とログイン機能をスポンサー側にも同じように実装
2. スポンサーという名前を事業者に変更
2. コミット&push
3. プルリク作成

## デザイン

シックで大人っぽいデザイン

## 仕様
異次元コミュニティ「HATAMOマッチングサービス」システムの 画面遷移図（簡易版テキスト）
1. ユーザー（利用者）視点 : localhost:3001
ログイン・新規登録ページ
トップページ → サービス一覧 / 検索
事業者詳細ページ → サービス内容 / 料金 / 申込ボタン
申込フォーム → 利用者情報入力 / 同意チェック
決済画面（Stripe） → 決済完了
完了画面 → 申込完了メッセージ
2. スポンサー（事業者）視点 localhost:3002
ログイン・新規登録ページ
スポンサー登録フォーム → 事業情報入力 / 必要書類アップロード
審査待ち画面 → 「運営審査中」ステータス
審査通過メール通知
決済画面（Stripe/アルファノート） → 月額料金支払い
管理画面トップ
　・案件一覧（利用者からの申込状況）
　・メッセージ通知（メール/LINE連動）
　・売上明細確認
3. 運営（管理者）視点 lcalhost:3003
管理者ログイン画面
スポンサー審査画面 → 承認 / 否認 / 再申請依頼
掲載管理画面 → 事業者リスト / 公開・非公開設定
申込管理画面 → 利用者申込一覧 / ステータス確認
精算画面 → 手数料控除後の振込額自動計算 / 明細PDF生成
:南京錠: 安全・法務導線
各フォームに reCAPTCHA
各購入・申込画面に キャンセル・返金規約同意チェック
フッターに 利用規約 / 特商法表示 / プライバシーポリシー
